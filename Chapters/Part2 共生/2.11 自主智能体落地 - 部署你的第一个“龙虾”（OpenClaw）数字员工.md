# 2.11 自主智能体落地：部署你的第一个“龙虾”（OpenClaw）数字员工

> [!IMPORTANT]
> **本章寄语**：写在纸上的兵书无法为你打赢战役，同样，保存在文件夹里的 `SKILL.md` 提示词也无法自动替你提炼文件。在这一章里，我们将使用 2026 年风靡全球的开源自主 Agent 框架——**OpenClaw（龙虾）**。我们将亲手在本地部署它，并把我们之前“蒸馏”出的核心 Skill 注入进去，让它真正具备在你的电脑上自动看文件、发通知的自主执行力。

在阅读《`Skill 蒸馏是个体永生的基石`》时，我们了解了如何把我们脑海中的“隐性知识”提取为规则清晰的 SOP（标准作业程序）。

但你肯定会问：“*这个写好的 SKILL.md 怎么用？难道每次都要我手动复制粘贴发给大模型吗？那还算什么自动化？*”

为了让 Skill 活过来，我们需要给它提供一个**运行环境（Runtime）**。
在 2026 年，最受极客和学生群体欢迎的开源智能体运行环境，就是 **OpenClaw（被网友亲切地称为“龙虾”或“龙虾 Agent”）**。

今天，我们就来亲手“养”一只龙虾，并把我们封装好的“认知灵魂”注入它的体内。

---

## 一、 认识你的数字员工：“龙虾”（OpenClaw）

**OpenClaw** 是一个旨在打破“聊天框屏障”的本地 Agent 执行框架。它的 Logo 是一只红色的龙虾，代表着它拥有像两只大螯（Claws）一样强大的**抓取、操控和执行力**。

它和普通的聊天机器人有本质区别：

```
普通 LLM 交互 ──> [人类提问] ──> [AI 动嘴回答] ──> [人类手动搬运数据]
OpenClaw 协同 ──> [给定目标] ──> [龙虾自主规划] ──> [调用本地工具/API] ──> [自动修改文件并提醒]
```

### 龙虾的核心能力：
1.  **操作系统操作员**：在授予权限后，它能自动检索你指定的文件夹，读取里面的文档、代码并进行修改。
2.  **网络冲浪手**：它能使用 headless 浏览器自动去搜索资料、下载 PDF 学术文献。
3.  **多端消息官**：它能自动把工作进展或整理好的错题大纲发送到你的飞书、钉钉、Telegram 或个人微信上。

## 二、 实战：在本地安装与运行 OpenClaw

别被“部署”两个字吓倒。跟着以下三步，在 5 分钟内你就能把它跑起来。

### 步骤 1：准备运行环境
1.  确保你的电脑已经安装了 `Node.js`（18+ 版本）或 `Docker` 容器管理器。
2.  获取 OpenClaw 客户端代码包（在终端中运行以下命令，或去其 GitHub 官网直接下载 Zip 解压包）：
    ```bash
    git clone https://github.com/openclaw/openclaw-runner.git
    cd openclaw-runner
    npm install
    ```

### 步骤 2：配置大脑（绑定 API Key）
要让龙虾聪明起来，它必须连接到大模型。
1.  在 `openclaw-runner` 文件夹下，复制 `.env.example` 文件并重命名为 `.env`。
2.  打开 `.env` 文件，填入你使用的大模型 API 密钥（推荐使用性价比极高的 DeepSeek R3/V5）：
    ```env
    OPENAI_API_BASE="https://api.deepseek.com/v1"
    OPENAI_API_KEY="你的-DeepSeek-API-密钥"
    DEFAULT_MODEL="deepseek-chat"
    ```

### 步骤 3：启动测试
在终端运行 `npm run dev`。当你看到终端显示一个可爱的红色龙虾 Logo，并提示 `OpenClaw is running on port 3000` 时，说明你已经成功激活了它的身体。

## 三、 灵魂注入：挂载你的第一个 `SKILL`

现在，我们要把在之前《`Skill 蒸馏`》里写好的“错题分析”或者“概念类比”Skill，绑定给这只龙虾。

1.  在 OpenClaw 的配置目录下，新建一个智能体配置文件：`agents/physics_assistant.yaml`。
2.  在文件中写入以下 YAML 配置代码，把你的 Skill 路径和执行权限关联起来：

```yaml
# 龙虾智能体卡片：物理提分官
name: "高考物理提分官"
description: "我的专属物理私教，能够读取本地错题并提取考点"

# 1. 灵魂注入：直接指向我们蒸馏好的 Skill Markdown 文件
system_prompt_file: "file:///i:/JZHO  NeuroBase/JZHO - Horizon/Chapters/Part2 共生/Skill 蒸馏是个体永生的基石.md"

# 2. 工具授权：允许这只龙虾使用哪些大鳌（能力）
tools:
  - read_local_directory   # 允许读取错题集目录
  - write_local_file        # 允许自动生成复习检查单
  - search_web              # 遇到不懂的问题允许上网搜索
  - send_telegram_notify    # 完成后自动发消息到我的手机

# 3. 目标设定：当启动它时它该干什么
default_task: "扫描错题目录，将最新添加的物理错题整理成复习清单并发送通知"
```

3.  启动运行！此时，龙虾会自动根据这个配置去读取你的物理错题文件夹。只要你把今天的错题拍照放进该文件夹，它就会自动按照你的“李雷版物理错题诊断”Skill 进行脑回路运算，并把最终的启发式解析推送到你的 Telegram 或微信上。

## 四、 养龙虾的“安全红线”与“防破产指南”

智能体拥有高度的本地权限，如果使用不当，它也可能带来灾难。因此，在配置文件中必须拉起两条坚不可摧的红线：

> [!WARNING]
> **智能体的两条警戒红线**
> 1.  **限额防破产（Token Spending Limits）**：智能体在后台自动规划时，如果遇到死循环，会疯狂调用 API。曾经有极客因为没有限制循环次数，一觉醒来大模型账单欠费数千元。请务必在 `.env` 或 Agent 的 yaml 中限制单次运行的最大 Token 消耗：
>     `max_daily_token_cost_usd: 1.5`（每天调用上限 1.5 美元）。
> 2.  **读写权限最小化（Principle of Least Privilege）**：绝对不要把你的 C 盘根目录授权给 OpenClaw！请专门为它建立一个名为 `AI_Workspace` 的空文件夹，只授权它在这个文件夹内进行读写，防止由于 AI 产生幻觉而误删你的系统文件或课件。
