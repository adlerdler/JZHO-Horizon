# 🎯 核心资源：行动清单汇总

> [!IMPORTANT]
> **本章寄语**：行动力是认知变现的唯一路径。本篇将《Horizon》全书一至四章的所有“本节行动检查清单”汇聚在一起。您可以将此清单复制到 Notion、嘀嗒清单，或者打印出来贴在桌前。每完成一项，就坚定地打上一个 `[x]`。

---

## 🧱 第一章：破壁 —— 搭建信息获取基建

*   **[1.1 信息围墙 - 你被困住了吗？]**
    *   [ ] 分析日常信息接触类型，识别偏见与过滤气泡。
    *   [ ] 严格限制推荐算法类 App（如刷短视频）单次连续使用时长在 15 分钟以内。
*   **[1.2 科学上网 - 打开全球知识大门]**
    *   [ ] 搭建或租用一条高速、隐私的境外网络连接。
    *   [ ] 测试对 GitHub、Wikipedia、Google 等核心学术站点的连通率与加载速度。
*   **[1.3 区块链通识 - 开启去中心化通行证]**
    *   [ ] 下载并配置 MetaMask 浏览器钱包或手机端。
    *   [ ] 手写并物理备份好你的 12 位助记词，绝不在联网设备中留存截图或文本。
*   **[1.3 英语突围 - 打开世界的钥匙]**
    *   [ ] 将所有主力电子设备（手机、电脑、IDE）的系统语言切换为英文。
    *   [ ] 安装智能双语网页翻译插件，开启每日 30 分钟英文技术博客阅读计划。
*   **[1.4 搭建第一个自己的博客站]**
    *   [ ] 在 GitHub 上新建博客仓库，配置好 Hexo / Hugo 等静态框架。
    *   [ ] 发表你的第一篇博客：《Hello World：我的破壁宣告》。
*   **[1.5 信息源管理 - 建立你的知识管道]**
    *   [ ] 下载 RSS 阅读器（如 NetNewsWire 或 Feedly）。
    *   [ ] 订阅包含 Hacker News、James Clear、大模型技术博客在内的 10 个以上高质量直达 RSS 源。
*   **[1.6 实战演练 - 从零搭建信息获取系统]**
    *   [ ] 部署完成你的自动聚合流，实现每天早晨 8:00 自动将精选科技雷达简报推送至你的 Telegram 或邮箱。

## 🤖 第二章：共生 —— 驾驭 AI 算力资产

*   **[2.1-2.2 大模型入门与助手配置]**
    *   [ ] 配置好 OpenAI API、Anthropic Claude 或国内高性价比大模型 API 额度。
*   **[2.3 提示词工程 - 与 AI 高效对话]**
    *   [ ] 练习编写包含 `Role, Context, Task, Constraints, Output Format` 五要素的 Markdown 结构化提示词。
*   **[2.4 AI 学习法 - 让 AI 成为你的私教]**
    *   [ ] 复制并运行“苏格拉底式私教”提示词，让 AI 带着你深度拆解一个复杂概念（如傅里叶变换、神经网络）。
*   **[2.7 认知跃迁 - 编写你的第一个 Skill]**
    *   [ ] 按照标准的 `SKILL.md` 格式，编写并沉淀一份你最擅长的流程（如“错题分析器.md”）。
*   **[2.8 自动化与工作流 - Coze 与 n8n 实操]**
    *   [ ] 在本地部署运行 n8n 容器，或在 Coze 平台搭建一个自动从特定文档抓取公式并答疑的微信/飞书机器人。
*   **[2.9 AI 编程启蒙 - 写出你的第一个应用]**
    *   [ ] 下载 Cursor 或 Trae 软件，用自然语言交互写出一个本地“精简待办清单”或“课表管理”网页应用。
*   **[2.11 自主智能体落地 - 部署你的第一个“龙虾”（OpenClaw）]**
    *   [ ] 克隆并本地跑通“龙虾（OpenClaw）”Agent，将你的专属 Skill 绑定给它，测试其自主执行日常报表统计的能力。
*   **[2.12 实战演练 - 搭建你的 AI 工作流]**
    *   [ ] 跑通“输入错题 $\to$ 工作流解析 $\to$ 龙虾生成复习卡片 $\to$ 推送微信”的完整本地智能自动化流程。

## 📈 第三章：觉醒 —— 一人公司（OPC）商业实践

*   **[3.1 一人公司 - 未来职场的终极形态]**
    *   [ ] 写下你的一人公司理念：明确你的数字军团构成和你提供什么非线性价值。
*   **[3.2 个人品牌 - 在公开中学习]**
    *   [ ] 在你的独立博客上写一篇文章，详尽披露你最近遇到的一个 Bug，以及你如何调试、解决它的全过程。
*   **[3.3 技能变现 - 你的第一桶金]**
    *   [ ] 撰写一份“AIGC定制化代劳”或“Chrome插件开发”的变现小草案，明确客户群体与微额定价。
*   **[3.4 时间资产 - 比钱更重要的东西]**
    *   [ ] 盘点你昨天的 24 小时：有多少时间被花在了“出卖肉体线性时间”上，有多少花在“构建可以无限复制的代码/文字资产”上？
*   **[3.5 商业思维 - 像企业家一样思考]**
    *   [ ] 模拟计算你当前小生意的单元经济模型。若没有生意，计算小华的案例。确保 LTV 能够大于 3 倍 CAC。
*   **[3.6 从大学开始 - 你的创业沙盒]**
    *   [ ] 设计一个手动代劳型（Concierge）MVP：不写一行代码，用手动加微信好友的方式完成一次需求闭环测试。
*   **[3.7 实战演练 - 设计你的个人商业模式]**
    *   [ ] 在白纸上画出九宫格，绘制并填满你的一人公司“个人商业模式画布”。

## 🔋 第四章：守恒 —— 个人能量与注意力管理

*   **[4.1 专注力危机 - 你正在失去什么]**
    *   [ ] 查看你昨天的屏幕使用时间、解锁次数及最常使用 App，关掉除电话/音视频通话外的所有 App 的推送红点。
*   **[4.2 数字极简 - 收回你的注意力]**
    *   [ ] 设置灰度（黑白）屏幕，将首屏图标彻底清空，把社交/娱乐 App 隐入应用库中。
    *   [ ] 学习时将手机静音并放置在 5 米外的包里，使用机械式番茄钟。
*   **[4.3 情绪管理 - 与自己和解]**
    *   [ ] 当你发现自己心烦意乱时，写下一份“CBT 情绪三栏表”（自动思维 $\to$ 认知偏差识别 $\to$ 理性重构）。
*   **[4.4 身体投资 - 你的第一资产]**
    *   [ ] 进行一次“无糖一日”实验，体验血糖平衡对专注度的影响。
    *   [ ] 在感觉脑雾严重时，立刻出门进行 20 分钟的中高强度有氧运动以激活 BDNF。
*   **[4.5 睡眠革命 - 被低估的超能力]**
    *   [ ] 设定一个固定的起床时间，以 90 分钟周期为单位向前倒推，严格执行睡前 45 分钟的“数字宵禁”。
*   **[4.6 长期主义 - 跑赢时间的秘密]**
    *   [ ] 挑选一项习惯（如编程或背词），设计一个 2 分钟内的极简启动微动作，并使用习惯叠加公式绑定旧习惯。
*   **[4.7 实战演练 - 设计你的能量管理系统]**
    *   [ ] 绘制你的“精力收支账户表”，制定并坚守你的 3 条精力底线红线规则。
