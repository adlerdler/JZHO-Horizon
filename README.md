## 📖 项目简介 | Overview

**JZHO-Horizon**（少年，你需要看到更大的世界）是一本专为有志向的青年打造的**开源个人成长与认知重构指南**。本项目旨在帮助你在喧嚣而充满不确定性的时代，打破认知局限，建立自我竞争优势：

- 🌐 **破壁**：突破信息茧房，建立全球视野
- 🤖 **共生**：拥抱 AI 时代，与技术共生进化
- 💡 **觉醒**：发现个人价值，创造商业可能性
- ⚡ **守恒**：管理自我能量，实现可持续成长

---

## ✨ 项目特色 | Features

### 📚 结构化内容
- 四章系统性内容，循序渐进（破壁 -> 共生 -> 觉醒 -> 守恒）
- 每章包含概念讲解、实践技巧和行动清单
- 章节间相互关联，形成网状知识结构

### 💡 实用性强
- 提供可操作的工具和落地方法论
- 包含大量模板和示例
- 实战演练帮助你边学边做，以行促知

### 🔗 知识互联
- 原生支持 Obsidian 双向链接
- 知识图谱可视化关系，构建个人第二大脑
- 提供核心概念索引表，便于速查

## 🚀 如何使用 | How to Use

### 方式一：Obsidian Vault（推荐体验最佳）
1. 下载安装 [Obsidian](https://obsidian.md/)
2. 将本项目的根目录作为 Vault（仓库）打开
3. 从 [序：在视界之外，重构你的世界](序：在视界之外，重构你的世界.md) 开始阅读，并体验知识图谱的魅力

### 方式二：Markdown 纯文本阅读
1. 使用任何你喜欢的 Markdown 编辑器（如 VSCode, Typora 等）
2. 或直接在 GitHub 网页端在线阅读
3. 结合整体文档树，从根目录开始逐级浏览各个章节内容

## 🤝 开源协作 | Contributing

**每个人都应该有机会看到更大的世界，而这个世界也需要你的参与。**

我们非常欢迎并鼓励任何人参与到这个开源项目中：

- 🐛 **报告问题**：发现错别字、链接失效或描述不准确，请提交 Issue。
- 💡 **提出建议**：有新的观点或希望补充的内容，欢迎在社区中交流。
- ✏️ **改进内容**：如果你对某一章节有更深的见解，欢迎直接提交 PR。
- 🌐 **多语言支持**：欢迎提供不同语言的翻译版本。

**参与流程：**
1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request
